# Preditor FCU - Base 29

Site estatico do Projeto Revelando Favelas para publicacao dos dashboards da base 29.

## Arquivos principais

- `index.html`: pagina inicial.
- `07_dashboard_ebm_avancado_areas_estudo_completo.html`: modelo completo.
- `07_dashboard_ebm_avancado_areas_estudo_morfologico.html`: modelo morfologico.
- `08_dashboard_metricas_modelos_areas_estudo.html`: analise dos modelos.
- `data_tiles/`: pontos do mapa em tiles comprimidos e poligonos FCU.
